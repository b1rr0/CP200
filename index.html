// Dijkstra O((V+E)logV) vector<vector<pair<int,int>>> g(n); vector<ll> dist(n,1e18); priority_queue<pair<ll,int>,vector<pair<ll,int>>,greater<>> pq; dist[0]=0; pq.push({0,0}); while(!pq.empty()){auto[d,v]=pq.top();pq.pop();if(d>dist[v])continue; for(auto[u,w]:g[v])if(dist[v]+w<dist[u]){dist[u]=dist[v]+w;pq.push({dist[u],u});}}

DSU — Union Find
10 lines of code, whole problem class · 124 CF
Kruskal = DSU + sort edges. Advanced: DSU with rollback for offline queries, weighted DSU for ranking/grouping.
// DSU path compression + union by rank vector<int> par(n),rnk(n,0); iota(par.begin(),par.end(),0); function<int(int)> find=[&](int x){return par[x]==x?x:par[x]=find(par[x]);}; auto unite=[&](int a,int b){a=find(a);b=find(b);if(a==b)return; if(rnk[a]<rnk[b])swap(a,b); par[b]=a; if(rnk[a]==rnk[b])rnk[a]++;};

💡 C++ Tips & Tricks for CP
Fast I/O
ios_base::sync_with_stdio(false); cin.tie(nullptr); — up to 5x faster. Never mix cin/cout with scanf/printf after this.
Lambda Comparator
sort(all(v), [](auto&a, auto&b){ return a.second > b.second; }); — custom sort in one line. Works with priority_queue too.
Structured Bindings
auto [x, y] = pii; and for(auto& [k, v] : map) — cleaner pair/tuple access (C++17).
Mod Arithmetic
(a % MOD + MOD) % MOD — safe for negative numbers. For division: use Fermat's little theorem a^(MOD-2) % MOD.
__builtin Functions
__builtin_popcount(x) — count bits. __builtin_clz(x) — leading zeros. __gcd(a,b). __lg(x) — floor(log2).
Custom Hash for unordered_map
CF has anti-hash tests! Use map or custom hash: struct H { size_t operator()(ll x) const { x ^= x >> 16; return x * 0x45d9f3b; }};
Overflow Guard
If n ≤ 10⁹, always use long long. Product of two ints? Cast first: (ll)a * b. Or 1LL * a * b.
STL Algorithms
*max_element(all(v)), accumulate(all(v), 0LL), unique(all(v)) after sort, next_permutation(all(v)), nth_element.
Pragmas (Codeforces)
#pragma GCC optimize("O2") — safe optimization. __int128 for huge numbers. Bitset can replace bool arrays for 64x speedup.
Useful Patterns
v.assign(n, 0) to reset. iota(all(v), 0) to fill 0..n-1. fill(dp, dp+N, -1) for memset. set<int,greater<>> for reverse order.
Binary Search on Answer
ll lo=0, hi=1e18; while(lo<hi){ ll mid=lo+(hi-lo)/2; if(check(mid)) hi=mid; else lo=mid+1; } — works for any monotone predicate.
Coordinate Compression
sort(all(v)); v.erase(unique(all(v)),v.end()); then index via lower_bound(all(v), x) - v.begin().
FastScanner
Always use BufferedReader + StringTokenizer (FastScanner). Scanner is 5-10x slower and will TLE on large input.
PrintWriter
PrintWriter out = new PrintWriter(new BufferedOutputStream(System.out)); — always out.flush() at the end. System.out.println is slow in loops.
Sorting Trick
Arrays.sort(int[]) uses quicksort — O(n²) worst case on CF anti-sort tests! Shuffle first or use ArrayList + Collections.sort() (mergesort).
Autoboxing Pitfall
Integer comparison: use .equals() or intValue(), not ==. == compares references for values > 127. Classic bug.
ArrayDeque > Stack/LinkedList
Use ArrayDeque for stack/queue — it's faster than Stack (synchronized) and LinkedList (pointer chasing). Java official recommendation.
TreeMap for Ordered Ops
map.floorKey(x), ceilingKey(x), firstKey(), lastKey() — equivalent of C++ lower_bound/upper_bound.
StringBuilder
Never concatenate strings in a loop: s += "x" is O(n) each time = O(n²) total. Use StringBuilder.append().
Long Overflow
int * int overflows silently! Cast first: (long)a * b. Use long for anything that can exceed 2·10⁹.
Bit Tricks
Integer.bitCount(x) — popcount. Integer.highestOneBit(x). Long.numberOfTrailingZeros(x). All O(1).
Lambda & Streams
Arrays.sort(arr, (a,b) -> a[1]-b[1]) — custom comparator. Avoid streams in CP (slow), but lambdas for comparators are fine.
Array Fill
Arrays.fill(dp, -1) for 1D. For 2D: for(int[] row : dp) Arrays.fill(row, -1); — no shortcut in Java.
Generic Pair Workaround
Java has no built-in Pair. Use int[] for speed, or define a minimal static class Pair<V,T> in your template. Map.Entry works too.
Fast Input
input = sys.stdin.readline — 3-5x faster than default input(). For bulk: data = sys.stdin.read().split() and iterate with index.
PyPy vs CPython
On CF, submit as PyPy 3-64 — 5-10x faster than CPython. If PyPy is unavailable, optimize I/O and avoid recursion.
Recursion Limit
sys.setrecursionlimit(300000) — default is 1000. Required for DFS on large graphs. Better: rewrite recursion as iterative stack.
defaultdict & Counter
from collections import defaultdict, Counter. defaultdict(int) — no KeyError. Counter(arr).most_common(k) — top-k in O(n log k).
heapq (Min Heap)
heappush(h, x), heappop(h). For max heap: push -x. For custom key: push tuples (priority, data).
bisect (Binary Search)
bisect_left(a, x) — lower_bound. bisect_right(a, x) — upper_bound. insort(a, x) — insert maintaining sort.
itertools
permutations(a), combinations(a, r), product(a, repeat=2), accumulate(a) — prefix sums in one line.
@lru_cache for DP
@lru_cache(maxsize=None) on recursive function = automatic memoization. Pass tuples, not lists (hashable). functools.cache in 3.9+.
SortedContainers
from sortedcontainers import SortedList — O(log n) add/remove, O(log n) index. Not in stdlib but available on CF. Python's set/dict equivalent.
List Comprehensions
[x*x for x in a if x > 0] — faster than loops. 2D grid: grid = [[0]*m for _ in range(n)]. Never [[0]*m]*n (shared refs!).
String Tricks
''.join(parts) — O(n) concat. s[::-1] — reverse. s.count(c). Strings are immutable — use list(s) for mutation.
Math
pow(a, b, MOD) — modular exponentiation (built-in, fast). math.gcd(a,b). Python ints have no overflow — unlimited precision.
